Export OutSystems UI CSS from your UI Kit

About the Plugin
How it works

The plugin access the various elements in the Sketch document and retreaves the values that it needs to build the base CSS and to show all the styles from the UI, like: colors, typography, borders, spacing, styles and others.

It also generates a second CSS with all the CSS variables that you may need in your project.

Don'ts
  • Do not change the name of the elements in Sketch, the plugin uses them to retrieve data to the CSS generator and changing them may cause the plugin to stop working.
  • Do not delete the base OutSystems UI elements from the pages or from the artboards, the plugin needs them to get data to the CSS generator and deleting them may cause the plugin to stop working.
  • Do not change the order the elements are organized in the tree of the artboard under penalty of the plugin stops working.
  • Avoid change the structure of the base OutSystems UI components. Since the plugin is expecting a specific structure to build the CSS, when changed, it may stop working.
Dos
  • Change the values of the base elements in the document (it will automatically update the CSS in the plugin).
  • Create new pages, artboards or elements.
  • Freely change or delete elements that were created by you.
